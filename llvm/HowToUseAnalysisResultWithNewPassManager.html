How to use analysis result with new pass manager in LLVM

LLVM has recently moved from legacy pass manager to new pass manager. However, there are only a few tutotials online that explains how to work with the new pass manager to write a pass. The official tutorials (using the new pass manager and writing an llvm pass) from LLVM gives a good start point, but it leaves a big gap for the reader who want to do more advanced analysis than a "Hello World" pass. In this post, I am going to cover how to use the analysis result by creating two passes - an analysis pass that count how many functions are in this module, and a readout pass that print the analysis result to stdout.
